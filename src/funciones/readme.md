Guía de ejercicios sobre Funciones en Python
En esta carpeta aprenderás a crear tus propias herramientas (funciones) para que tu código sea más ordenado y no tengas que repetir lo mismo muchas veces.

Orden recomendado de lectura / ejecución
Para entender bien las funciones, te recomiendo seguir este orden:

01_definicion_basica.py: Empieza por aquí para ver cómo se crea una función y cómo se llama.
02_parametros_argumentos.py: Aprende a pasarle datos a tus funciones de diferentes formas.
03_valor_retorno.py: Mira cómo hacer que tus funciones te devuelvan resultados útiles.
04_documentacion_docstrings.py: Descubre cómo explicar qué hace tu código para no olvidarlo después.
Cómo ejecutar cada ejercicio
Puedes probar cada archivo usando el comando python desde la terminal en la raíz del proyecto.

Ejemplo:

python src/funciones/01_definicion_basica.py
¡Espero que estos ejemplos te ayuden a dominar las funciones en Python!
