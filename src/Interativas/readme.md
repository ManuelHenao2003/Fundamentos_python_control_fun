Guía de ejercicios sobre Estructuras Iterativas (Bucles)
En esta carpeta encontrarás ejemplos prácticos para aprender cómo repetir tareas en Python de forma automática.

Orden recomendado de lectura / ejecución
Para aprender bien cómo funcionan los bucles, te sugiero seguir este orden:

01_bucle_for_range.py: Empieza por aquí para ver cómo recorrer listas y usar la función range.
02_bucle_while.py: Aprende a usar bucles que dependen de una condición que puede cambiar.
03_break_continue.py: Descubre cómo saltarte pasos o salirte de un bucle antes de tiempo.
04_pass_else_bucles.py: Mira estos casos especiales para dejar huecos en el código o hacer algo al terminar un bucle.
Cómo ejecutar cada ejercicio
Para ejecutar los archivos, abre una terminal en la carpeta principal del proyecto y escribe python seguido del nombre del archivo que quieras probar.

Por ejemplo, para el primer tema:

python src/iterativas/01_bucle_for_range.py
¡Espero que te sirvan mucho estos ejemplos para practicar!
