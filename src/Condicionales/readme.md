Guía de ejercicios sobre Condicionales
En esta carpeta encontrarás varios ejemplos para aprender cómo controlar el camino que sigue un programa en Python usando condiciones.

Orden recomendado de lectura / ejecución
Para aprender paso a paso, te recomiendo seguir este orden:

01_declaracion_if.py: Aprende lo más básico, el uso del if.
02_declaracion_if_else.py: Qué pasa cuando hay dos opciones (si se cumple o no).
03_declaracion_if_elif_else.py: Cómo manejar muchas opciones seguidas.
04_declaracion_match_case.py: Una forma moderna de comparar una cosa contra varias posibilidades.
05_operadores_logicos.py: Uso de and, or y not para condiciones más complejas.
06_condicionales_anidados.py: Cómo poner un if dentro de otro.
07_expresiones_condicionales.py: Cómo escribir un if-else en una sola línea.
08_evaluacion_cortocircuito.py: Trucos para que Python trabaje menos y el código sea más seguro.
Cómo ejecutar cada ejercicio
Para probar los ejemplos, asegúrate de estar en la carpeta raíz del proyecto y usa el comando python seguido de la ruta del archivo.

Ejemplo para ejecutar el primer tema:

python src/condicionales/01_declaracion_if.py
Puedes ir cambiando el nombre del archivo para probar los demás. ¡Disfruta aprendiendo!
